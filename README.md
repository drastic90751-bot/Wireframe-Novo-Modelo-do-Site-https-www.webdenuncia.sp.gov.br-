# Wireframe-Novo-Modelo-do-Site-https-www.webdenuncia.sp.gov.br-
Este artefato apresenta um wireframe de baixa fidelidade do site Web Denúncia SP, desenvolvido na disciplina de Interface e Jornada do Usuário da UNICID. O modelo prioriza clareza, reduz carga cognitiva e melhora a navegação, corrigindo falhas como fragmentação e falta de marcos.
